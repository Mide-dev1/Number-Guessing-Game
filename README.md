# Number Guessing Game (Python)
A simple command-line number guessing game built with Python.
The program generates a random number, and the user tries to guess it with feedback provided after each attempt.

## Features

* Random number generation
* User input handling
* Feedback system (Too high / Too low)
* Attempt counter
* Loop-based gameplay until correct guess

## What I Learned

* Using "while" loops for continuous interaction
* Applying conditional statements ("if", "elif", "else")
* Working with comparison operators
* Using the "random" module
* Writing interactive command-line programs

## How to Play

1. Run the Python file
2. Enter a number when prompted
3. The program will tell you if your guess is too high or too low
4. Keep guessing until you get the correct number
